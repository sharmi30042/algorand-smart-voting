# Algorand Smart Voting System

## Overview
This repository contains my Algorand blockchain project developed as part of the course assignment.

The goal of this project is to explore decentralized applications (dApps) using the Algorand blockchain platform.

## Project Idea
The project will implement a **secure blockchain-based voting system** where votes are recorded transparently using Algorand smart contracts.

## Features
- Secure digital voting
- Blockchain-based transparency
- Tamper-proof vote storage
- Decentralized architecture

## Technologies
- Algorand Blockchain
- Smart Contracts
- GitHub for version control
